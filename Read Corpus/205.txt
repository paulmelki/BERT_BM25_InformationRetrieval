arecales is an order of flowering plants the order has been widely recognised only for the past few decades until then the accepted name for the order including these plants was principes taxonomy the cronquist system of assigned the order to the subclass arecidae in the class liliopsida monocotyledons the thorne system and the dahlgren system assigned the order to the superorder areciflorae also called arecanae in the subclass liliidae monocotyledons with the single family arecaceae the apg ii system of recognised the order and placed it in the clade commelinids in the monocots and uses this circumscription order arecales family arecaceae alternative name palmae this was unchanged from the apg system of although it used the spelling commelinoids instead of commelinids the apg iv system of places dasypogonaceae in this order after studies shown dasypogonaceae as sister to arecaceae however this decision has been called into question references external links ncbi taxonomy browser
